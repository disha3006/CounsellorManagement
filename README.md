Counsellor Management System: A web-based application built using Spring Boot, Thymeleaf, MySQL, and Bootstrap, designed to manage counsellors and student enquiries efficiently.

Features
-User (Counsellor) Module
Counsellor login/logout
Role-based access (ADMIN / COUNSELLOR)
Dashboard showing:
Total enquiries
Open enquiries
Enrolled students
Lost enquiries

-Enquiry Module
Add new enquiry
Edit/update enquiry
List all enquiries (pagination supported)
Filter by status (Open, Enrolled, Lost)
Dashboard analytics for counsellors

-Admin Module
Admins can:
Add counsellors
Update counsellor info
Delete counsellors
View all counsellors

-Tech Stack
Layer	                 Technology
Backend	               Spring Boot (MVC, Data JPA, Security optional)
Frontend             	 Thymeleaf + Bootstrap 5
Database	             MySQL
Build Tool	           Maven
Server	               Embedded Tomcat
